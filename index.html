Cours GIT

Qu'est ce que GIT ?

GIT est un outil de versionning, et permet d'avoir une traçabilité et un historique de tous les fichiers du projet.

Il permet également de travailler en équipe: envoyer nos modifications à nos collegues et récuperer nos modifications des collegues.

Dans la traçabilité des fichiers, GIT nous donne toujours 3 info: Qui ? Quand? Pourquoi ?

Les commandes de base

Initialiser un projet:
git init
Voir le statut des fichiers
git status
Ajouter des fichiers
git add 'fichier'
Ajouter TOUS LES FICHIERS :
git add .
Commit (enregistrer) les fichiers ajoutés
git commit
Un commit necessite toujours un message
commit avec message:
git commit -m "message'
commit avec add + message (fonctionne sur les fichers déjà add précedemment):
git commit -a -m "message"
